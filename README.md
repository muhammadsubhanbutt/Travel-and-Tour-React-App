# 🌍 Travel and Tour React Application

A modern and responsive Travel and Tour website built using **React** and **Vite**. This project is a static frontend application that showcases popular travel destinations, recent trips, and provides users with a simple contact and signup interface.

##  Live Demo

🔗 https://travel-and-tour-react-app-coral.vercel.app/

---

##  About the Project

This project is a responsive travel website developed using React. It demonstrates component-based architecture, React Router navigation, responsive layouts, and modern UI design.

The website includes multiple pages such as Home, About, Services, Contact, and Signup, making it a complete frontend project suitable for learning React fundamentals.

---

##  Features

- Responsive design for desktop, tablet, and mobile devices
- Beautiful Hero Section
- Popular Destinations Section
- Recent Trips Cards
- About Page
- Services Page
- Contact Form UI
- Signup Page UI
- Responsive Navigation Bar with Hamburger Menu
- Footer with Social Media Icons
- Component-based architecture
- Client-side routing using React Router

##  Installation

Clone the repository

```bash
git clone https://github.com/muhammadsubhanbutt/Travel-and-Tour-React-App.git
```

Move into the project

```bash
cd Travel-and-Tour-React-App
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:5173
```

---

##  Build for Production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
